# movable_window
**用C++实现控制窗口移动**

## 核心功能
```窗口移动控制```
- WASD键控制移动
- 方向键（↑↓←→）控制移动
- 支持斜向移动（同时按两个键）
---
```智能重置系统```
- 空格键：手动立即重置到屏幕中心
- ESC键：取消自动重置标记（不退出程序）
- 自动重置：窗口完全移出屏幕5秒后自动复位

### 界面特点
- 现代化深色UI
- 圆角设计和阴影效果
- 实时按键状态反馈
- 中央十字方向键布局
- 按钮有按下/释放状态显示
- 实时位置坐标显示
- 状态信息显示
- 操作提示信息

####技术特性
- 流畅控制
- 50ms刷新率确保响应灵敏
- 双缓冲绘图消除闪烁
- 平滑移动（每帧5像素）
- 智能边界处理
- 精确的边界检测逻辑
- 资源优化
- 正确管理GDI对象
